Cake Defense: Ant Rush
A mobile-friendly wave defense. Protect the cake in the center. Ants rush in from all sides.

Controls
- Tap an ant to eliminate it.

Rules
- Wave 1 has 10 ants.
- Each ant that reaches the cake removes 1 HP.
- Cake starts with 3 HP.
- Clear all ants to finish the wave.
- If HP hits 0, game over.

Install
- Host the folder on Netlify, Vercel, or GitHub Pages.
- Open on your phone, then Add to Home Screen.
- Works offline after the first load.
